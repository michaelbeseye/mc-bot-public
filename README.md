# MC-Bot

mc-bot 是一套預先編寫好並可以輕鬆投放至你的多人世界裡的 Bot 玩家。他能夠為你完成各項簡易到複雜的任務，包含護送、搬運、駐守，以及偶爾跟你講講幹話：）  
mc-bot 是基於 [PrismarineJS Mineflayer](https://github.com/PrismarineJS/mineflayer)  所創造的應用，並遵守 MIT 授權條款。
